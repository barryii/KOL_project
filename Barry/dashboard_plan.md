# 執行計畫：YouTuber 比較儀表板架構與體驗優化

本計畫記錄儀表板從初始版本到「全域連動」版本的各項架構升級與優化設計。

## 1. 全域數據連動與視覺對齊 (最新優化)
*   **全域篩選維度 (Global Analytics Switcher)**：將原本分散於各區塊的影片類型篩選器整合至 Header，並透過全域狀態 `globalVideoType` 同步驅動所有分頁之數據。
*   **專業圖表 API 設計**：`/api/top_videos` 支援透過 `top_n=0` 進行「不限數量（Unlimited）」的數據抓取，確保前端能緩存全量歷史影片進行瞬時過濾。
*   **圖表座標軸對位 (Vertical Alignment)**：為解決數值級距造成的圖表偏移，強制鎖定趨勢圖的 Y 軸標籤寬度（80px），使不同量級圖表的左側繪圖格線完美垂直對齊。
*   **跨圖表同步互動 (Synced Hover & Tooltips)**：游標懸停時，透過 `syncChartsTooltip` 廣播 X 軸索引，使所有相關趨勢圖同步顯示數據提示框與垂直虛線，提供跨維度的瞬間對比。

## 2. 鐵粉排行榜 API 重構 (已完成)
*   **專業級 API 升級**：將 `/api/top_videos` 重構為支援 `top_n=0` 無限制抓取。
*   **全站全量更新**：前端已改為請求全量影片 (top_n=0)，並補強了提示框殘影問題。
*   **前端即時排序**：透過下拉選單動態在 HTML 前端過濾，並切換「留言數 / 獲愛心數」的精準排序。

## 3. 堆疊直條圖與版面優化 (已完成)
*   **滿版網格**：放大圖表顯示區域（解決原本側邊擠壓問題），提升資訊識別度。
*   **組成成分拆解**：將總量數據（觀看、互動）精準拆分為「一般影片、Shorts、直播」的迷你堆疊圖 (Stacked Bar Charts)，讓內容貢獻度一目了然。
*   頁面頂部新增了膠囊狀的控制組：**🎬 全部**、**📹 一般影片**、**⚡ Shorts**、**🔴 直播**。
*   後端已升級為「不設限 API (`top_n=0`)」，現在您只要在頂部切換，全站歷史影片都會瞬間反應。
