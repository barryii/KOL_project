前端與 API 介接

透過 FastAPI 處理前端儀表板的請求，即時撈取各頻道健康度、情緒分析與社群鐵粉清單的統計結果。

AI 與資料管線

  • NLP 分析：處理留言並標註情緒分數與意圖
  • 機器學習：影片內容屬性自動 Cluster 分群
  • 社群聚合：計算 Top N 鐵粉排行

主從資料庫架構

主資料庫負責所有寫入操作，透過 Replication 即時同步至備援資料庫 (Read Replica)，讀寫分離提升可用性與查詢效能。

YouTube Data API v3

由排程任務定時觸發，透過官方 API 擷取影片清單、留言、頻道統計等原始數據並寫入資料庫。

核心基礎資料表

channels 與 videos 儲存從 YouTube API 擷取的基礎指標。videos 表含 cluster_label 欄位，用於歸類影片屬性。

AI 擴充欄位

在 video_comments 中，除原始文字外皆有 AI 處理後的擴充標籤：

  • sentiment: 正向 / 負向 / 中立
  • sentiment_score: 1–5 分量化
  • topic_tag: 意圖標籤

核心鐵粉分析表

topN_comments 系列資料表彙總高頻率互動的 author_id，計算 comment_count 與 total_likes，精準識別忠實 VIP 觀眾。
